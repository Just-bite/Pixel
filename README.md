# Pixel - Фоторедактор

![C++](https://img.shields.io/badge/C++-17-blue.svg)
![Qt](https://img.shields.io/badge/Qt-5.0-green.svg)

Редактор изображений с расширенным функционалом, разработанный на C++ с использованием Qt Framework.

## 🌟 Возможности

### **Управление слоями**
- **Иерархическая система слоев** 
- **Растровые  маски** для фильтров

### **Объекты и векторная графика**
- **Текстовые объекты**:
  - Полная поддержка системных шрифтов
  - Настройка размера
  - Цвет текста
- **Геометрические фигуры**:
  - Прямоугольники, эллипсы
  **Изображения**:
  - Вставка и перетаскивание.
- **Пространственные преобразования**:
  - Поворот
  - Масштабирование

### **Матричные фильтры**
- **Размытие**: Гауссово
- **Резкость**: Unsharp Mask
- **Пороговая фильтрация**

### **Цветовые пространства**
- Поддержка RGBA
- Инверсия
- Настройка яркости, контраста, насыщенности

### **Базовые инструменты**
- **Карандаш** с настраиваемой толщиной и жесткостью
- **Заливка** (Flood Fill) с допуском
- **Ластик** с мягкими краями
- **Пипетка** для захвата цвета
- **Фигура** (эллипсы и прямоугольники)
- **Текст** для текста
- **Стрелочка** для указания на объект
- **Ладошко** для перемещения по канве

### **Стек изменений**
- Полная история действий с использованием паттерна "команда"
- Сохранение истории в проекте
- Возможно откатить растеризацию

### **Работа с файлами**
- **Экспорт**: PNG
- **Импорт**: Поддержка 50+ форматов через Qt
- **Собственный формат проекта** (.pxl) с сохранением:
  - Всех слоев и их свойств
  - Объектов и трансформаций
  - Настроек проекта
  - Конфигурация фильтров

## **Пользовательский интерфейс**

### Ключевые элементы UI
- **Горячие клавиши** как в Photoshop
- **Палитра цветов** с возможностью непосредственного ввода(HEX) или HSI цветовое пространство и alpha
- **Информационная панель** с уведомлениями о статусе проекта и перечне транзакций(экспорт), а также кнопки управления масштабом
- **Панель инструментов** выбор инструмента по его иконке с интуитивно понятными вспылвающими контекстными подсказками
- **Контекстная панель** динамическая настройка параметров выбранного нструмента или объекта или не не фильтра
- **Панель слоев** для настройки слоев для добавления слоев для добавления филттров и удаления фильтров и слоев и перемещение их можно
- **Рабочая поверхность** непосредственно основное рабочее пространство для работы 

## 🚀 Начало работы

### Системные требования
- **ОС**: Windows 10+,
- **Компилятор**: GCC 9+, Clang 10+, MSVC 2019+
- **Qt**: версия 5.0 или выше

### Сборка из исходников

```bash
# Клонирование репозитория
git clone https://github.com/Just-bite/Pixel.git
cd Pixel

# Создание директории сборки
mkdir build && cd build

# Генерация проектных файлов
cmake .. -DCMAKE_BUILD_TYPE=Release

# Сборка
cmake --build . --config Release

# Запуск
./pixel
```
